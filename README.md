# 학생 주제탐구 가이드

학생들의 주제탐구를 돕는 사이트와 활용 팁 모음을 제공하는 진로도우미 사이트'입니다.

## 🔹 기능
- 대학 홈페이지, 논문 사이트, 학술 검색, 강의 사이트, AI 도구, 발표자료, 교과서 등 카테고리별 자료 제공
- 각 사이트에 대한 설명과 바로가기 링크 제공
- 탐구 주제 자료 조사와 발표 준비 지원
